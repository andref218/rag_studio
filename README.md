# RAG Studio

RAG Studio is an advanced Retrieval-Augmented Generation (RAG) platform that enables users to chat with multiple PDF documents while providing complete transparency into the retrieval process.

The project focuses on building a high-quality RAG pipeline using open-source embedding models, optimized chunking strategies, and retrieval evaluation metrics such as MRR, Recall@K, Precision@K, and NDCG.

Designed as both a practical document assistant and an AI engineering experimentation platform, RAG Studio allows developers to benchmark different retrieval configurations and understand how each design choice impacts RAG performance.
